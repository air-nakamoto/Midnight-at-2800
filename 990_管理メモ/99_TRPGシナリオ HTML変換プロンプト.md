# TRPGシナリオ HTML変換プロンプト

整理済みのシナリオフォルダを、指定されたテンプレートを用いて単一HTMLに変換してください。

📁 入力: [シナリオフォルダパス]
📄 テンプレート: ideal_template.html
📄 出力: [シナリオ名]_session.html

---

# 変換ルール

1. **テンプレートの読込**: 同ディレクトリにある `ideal_template.html` の内容をベースにします。
2. **サイドバーの生成 (`#file-tree`)**:
   - 入力フォルダの構造に従って、`div.tree-item` を生成し `#file-tree` 内に配置してください。
   - フォルダは `div.tree-item.has-children` と `div.tree-children` で入れ子構造にします。
   - `Explorer` などのヘッダー部分はテンプレートのまま維持します。
   - 各ファイル項目には `onclick="openPage('sec-[ファイル名]', this)"` を付与します。
3. **コンテンツの生成 (`.content-area`)**:
   - 各テキストファイルの内容を読み込み、`.page` クラスを持つ `div` として `.content-area` 内に生成します。
   - IDはサイドバーのリンクと一致させます (`id="sec-[ファイル名]"`）。
   - マークダウン記法（見出し、リスト、太字など）をHTMLに変換します。
   - テキストファイル内の `[script]...[/script]` やダイス表記等の独自記法があれば、テンプレートの `.script-box` や `.card` 等に適宜変換します。

# デザイン・機能仕様 (テンプレート準拠)

以下の機能はテンプレート (`ideal_template.html`) に実装済みのため、**HTML構造やJS/CSSを改変しないでください**。

- **テーマ切替**: ダーク/ライトモード
- **ツリー操作**: 全展開 / 全折畳
- **ナビゲーション**: トップへ戻るボタン
- **デザイン**: 異変表テーブル (`.table-custom`), バッジ (`.badge`), カード (`.card`)

# 禁止事項

- **チェックボックスの追加**: 進行管理チェックボックス機能は廃止されました。追加しないでください。
- **パンくずリストの追加**: パンくずリストは廃止されました。追加しないでください。
- **設定画面の生成**: 強調表示設定などのモーダルやすべての設定機能は不要です。
