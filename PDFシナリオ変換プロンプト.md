# PDFシナリオをテンプレートに変換するプロンプト

以下のプロンプトをChatAIに渡すことで、PDFシナリオをこのテンプレート構造に変換できます。

---

## プロンプト本文
```
以下階層に存在するPDFは、TRPGシナリオです。
/Users/air/Documents/Air Date/TRPG/CoC/シナリオ/ペケペケ.pdf

このシナリオを以下のフォルダ・ファイル構造に沿って整理してください。
上記のフォルダに作成してください

# フォルダ構造

## 補助テキストファイル（ルートディレクトリ・最上位表示）
- 00_WP前提.txt（前提条件、推奨人数、プレイ時間など）
- 01_概要.txt（シナリオ全体の概要）
- 02_ハンドアウト.txt（各PCへのハンドアウト）
- 03_PC情報.txt（PC1〜PC5の情報）
- 04_ナチュラルプランニング.txt（企画意図、目的、ビジョン）※空でも作成
- 05_案.txt（アイデアメモ）※空でも作成

## メインフォルダ（シナリオの流れ）
- 000_導入/
  - 00_導入_概要.txt（導入フェイズ全体の概要）
  - 01_導入1-1.txt（最初の導入シーン）
  - 02_導入1-2.txt（2番目の導入シーン）
  - ... 必要に応じて追加

- 001_ミドル1/
  - 00_ミドル1_概要.txt（ミドル1フェイズ全体の概要）
  - 01_ミドル1-1.txt（探索箇所やイベント1）
  - 02_ミドル1-2.txt（探索箇所やイベント2）
  - ... 必要に応じて追加

- 002_ミドル2/（必要に応じて）
- 003_ミドル3/（必要に応じて）

- 099_クライマックス/
  - 00_クライマックス_概要.txt
  - 01_CLX1.txt（クライマックスルート1）
  - 02_CLX2.txt（クライマックスルート2、複数ある場合）

- 100_エンディング/
  - 00_エンディング_概要.txt
  - 01_ED1.txt（エンディング1）
  - 02_ED2.txt（エンディング2、複数ある場合）

- 101_報酬/
  - 00_報酬_概要.txt（報酬、SAN値回復、技能上昇など）

- 102_解説/
  - 00_解説_概要.txt（シナリオの裏設定、GM向けの補足情報）

# 変換ルール

1. **シナリオの構造を分析**
   - 導入、ミドル、クライマックス、エンディングの区切りを特定
   - ミドルフェイズが複数ある場合は、001_ミドル1、002_ミドル2...と分割

2. **各ファイルの内容**
   - 00_〇〇_概要.txt：そのフェイズ全体の流れや目的を簡潔に
   - 01以降のファイル：具体的なシーン、探索箇所、イベント、NPCとの会話など

3. **番号の割り振り**
   - **補助ファイル**: 00〜05で始まる（エクスプローラーで最上位表示）
   - **フォルダ番号**: 000から開始、クライマックスは099、エンディングは100
   - ミドルの数が多い場合は004、005...と増やす
   - 各フォルダ内のファイルは00（概要）から開始し、01、02...と続ける

4. **情報の整理**
   - SAN値チェック、判定、技能使用などはそのまま記載
   - NPC情報は該当シーンに記載し、まとめも102_解説/に
   - 報酬は101_報酬/にまとめる

5. **出力形式**
   - 各ファイルの内容をマークダウン形式で出力
   - ファイル名を明記した上で、内容を記載
   - 例：
```
     ## 00_WP前提.txt
     【想定プレイ人数】3〜4人
     【想定プレイ時間】4〜5時間
     ...

     ## 01_概要.txt
     このシナリオは...

     ## 000_導入/00_導入_概要.txt
     導入フェイズでは、探索者たちが...
```

6. **不明な部分**
   - PDFから明確に判断できない場合は、最も適切と思われる場所に配置
   - 配置理由を簡潔にコメントとして記載

# 注意事項
- 補助ファイル（00〜05）は必ず番号付きで作成し、エクスプローラーで最上位に表示されるようにする
- 04_ナチュラルプランニング.txt と 05_案.txt はPDFに該当情報がなければ空ファイルとして作成
- ミドルの数はシナリオによって異なります（0個〜多数）
- 探索フェイズが長い場合は、複数のミドルフォルダに分割してください
- クライマックスやエンディングが1つの場合でも、フォルダ構造は維持してください
```
